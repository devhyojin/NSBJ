## Social Login(Google)

``` mermaid
sequenceDiagram;
	
   	participant User
    participant Landing Page
    participant OAuth2.0

    User->>Landing Page : 구글 로그인 버튼 클릭
    Landing Page ->>OAuth2.0: 회원정보 전송
    OAuth2.0 ->>OAuth2.0: 유저 정보 검증
    OAuth2.0 ->>Landing Page: googel_id / Token 제공
    Landing Page ->>User: 메인페이지 제공
```

## Main page

``` mermaid
sequenceDiagram;

    participant User
    participant Main Page
    participant Spring Boot
    participant WebAPI / KAKAO_API
    participant MODE

        Main Page ->>MODE:현재 모드 정보 요청
        MODE ->>Main Page:라이트/다크모드(일몰시간)
        User ->>Main Page:버튼클릭(위치등록)
        WebAPI / KAKAO_API ->>Main Page:사용자 현재 위치정보 제공
        Main Page ->>Spring Boot: 위치정보 전달
        Spring Boot ->>Main Page: 동네 사용자 및 채팅방 정보 전달
        Main Page ->>User:채팅방 활성화 및 주변 사용자 시각화
```

## My page
``` mermaid
sequenceDiagram;

    participant User
    participant My Page
    participant Spring Boot
    participant MODE

    My Page ->>MODE:현재 모드 정보 요청
    MODE ->>My Page:라이트/다크모드(일몰시간)
    My Page ->>User: localStorage Userinfo 제공
    User ->>My Page: 뱃지/닉네임/캐릭터 변경 요청
    My Page ->>Spring Boot: 사용자 변경 요청
    Spring Boot ->>Spring Boot: DB수정
    Spring Boot ->>My Page:변경완료 및 데이터 전송
    My Page ->>User: 변경 반영
```

## Chat

sequenceDiagram;
